## Recommendations

### Immediate (Free)

1. **Search USPTO TESS Directly**
   - URL: https://tmsearch.uspto.gov/
   - Search each name in Class 9 (software) and Class 42 (SaaS)
   - Screenshot results for records

2. **Check App Store**
   - Search "SaneBar", "SaneClip", etc.
   - Document any conflicting apps

3. **Search State Trademarks**
   - California: https://businesssearch.sos.ca.gov/
   - Delaware (common for LLCs): https://icis.corp.delaware.gov/

### Short-Term ($250-500)

4. **File Intent-to-Use Trademark**
   - Can file before product launches
   - $250-350 per class per mark
   - Recommended order: SaneApps first (umbrella), then individual apps

5. **Consider LLC Formation**
   - "Sane Apps LLC" or "MrSane LLC"
   - Delaware or Wyoming (privacy-friendly)
   - $100-300 to form

### Long-Term

6. **Trademark Attorney Consultation**
   - For SaneApps name (due to potential conflict)
   - ~$300-500 for initial consultation
   - Can do comprehensive search

## Trademark Classes for Software

When filing, use these classes:

- **Class 9**: Computer software; downloadable software
- **Class 42**: Software as a service (SaaS); cloud computing

For macOS desktop apps, Class 9 is primary.
